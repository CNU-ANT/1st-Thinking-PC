개미굴 
 
 성실하게 일을 하는 개미는 계층적으로 집을 만드는 걸로 유명하다. 
 우리의 천재 공학자 현종이는 어느날 과일을 잘라 옮기는 특별한 개미들을 보고 이 개미에 대해 흥미가 생겼다.
 현종이는 해당 개미의 습성에 대해 연구하면서 개미굴을 탐험하고 어떤 과일이 저장되어있는지 알려주는 로봇 개미를 만들었다. 
 다음은 현종이가 알아낸 특별한 개미의 습성이다. 
 1. 이 개미는 오직 과일들만 저장을 한다.
 2. 각 계층별로 저장을 한다. 
 3. 갈래길에도 과일을 쌓아 놓는다.
 4. 해당 저장소로 내려 가는 길은 하나다.
 
로봇 개미는 다시 뒤돌지 않기 때문에 가장 깊은 곳까지의 정보만을 준다.
현종이는 개미굴의 모습을 추론해보기 위해 여러개의 개의 로봇 개미를 넣었다.

다음은 4개의 로봇 개미를 넣었을 때 로봇 개미들이 보내준 개미굴 정보다. 
KIWI / BANANA
KIWI / APPLE
APPLE / APPLE
APPLE / BANANA / KIWI
 
 '/'를 기준으로 왼쪽으로 갈 수록 깊은 곳에 위치한 과일들이다. 이 정보를 가지고 다음과 같이 개미굴의 모양을 시각화하면 다음과 같다. 
 
APPLE
--APPLE
--BANANA
----KIWI
KIWI
--APPLE
--BANANA

 각 깊이는 "--" 로 구분이 된다. 같은 깊이 일 때는 사전 순서대로 정렬이 되며 깊이 우선으로 표시를 한다.
 현종이는 이 시각화 작업을 우리에게 외주로 부탁했다. 
 이제 이 데이터를 바탕으로 개미굴이 어떤 식으로 되어있는지 표현해보자 

 
입력:
N: 모든 디렉토리의 경로의 개수
디렉토리 구조는 / 를 기준으로 오른쪽이 왼쪽의 하위 디렉토리를 의미한다.
출력:
같은 깊이의 디렉토리는 사전순으로 출력하며,
어느 디렉토리의 하위디렉토리일 경우 -- 를 붙인다.

예제 입력:
3
B / A
A / B / C / D
A / C
예제 출력:
A
--B
----C
------D
--C
B
--A

예제 입력:
3
KIWI / BANANA
KIWI / APPLE
APPLE / APPLE
APPLE / BANANA / KIWI

예제 출력:
APPLE
--APPLE
--BANANA
----KIWI
KIWI
--APPLE
--BANANA
