디렉토리 트리
영학이형은 파일 search 엔진을 만들려고 한다.
파일 search엔진은 모든 파일의 경로를 저장하고 있어서
매우 빠르게 원하는 파일이 어디있는지 알 수 있다.
모든 디렉토리의 경로가 주어졌을 때
각 디렉토리의 순서를 트리 형태로 출력하라.

입력:
N: 모든 디렉토리의 경로의 개수
디렉토리 구조는 / 를 기준으로 오른쪽이 왼쪽의 하위 디렉토리를 의미한다.
출력:
같은 깊이의 디렉토리는 사전순으로 출력하며,
어느 디렉토리의 하위디렉토리일 경우 -- 를 붙인다.

예제 입력:
3
B / A
A / B / C / D
A / C
예제 출력:
A
--B
----C
------D
--C
B
--A

예제 입력:
3
KIWI / BANANA
KIWI / APPLE
APPLE / APPLE
APPLE / BANANA / KIWI

예제 출력:
APPLE
--APPLE
--BANANA
----KIWI
KIWI
--APPLE
--BANANA
