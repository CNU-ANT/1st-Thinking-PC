선형 회귀

문제
시험 결과가 나온 후, 모든 학생들의 공부시간과 시험성적을 조사하였다.

(그림 삽입: 2차원 좌표계에 각 학생에 대해 (공부시간, 시험성적) 좌표에 점을 찍음, line fitting f(x) = ax + b, 좋은 line과 나쁜 line을 그린다)

이 데이터로부터, 공부시간 x가 주어졌을 때 시험점수 y를 예측하는 일차함수 f
y = f(x) = ax + b
를 구하고 싶다. 그림으로부터, g(x)=2x+3이 h(x)=4x+5보다 좋은 예측함수라는 것을 직관적으로 알 수 있다. 좋은 예측함수란 제곱합오차(square sum error)

(제곱합오차 식 삽입)

가 작은 예측함수이다. 제곱합오차를 최소로 하는 a와 b를 구하여라. (단, a와 b는 모두 100 이하의 양의 정수이다)
오버플로우에 주의하여라.

입력
첫째 줄에는 데이터의 수 N이 주어진다. (2 ≤ N ≤ 100)
둘째 줄부터 N개의 줄이 더 주어진다. 각 줄에는 i번 학생의 공부시간 x_i와 시험성적 y_i가 순서대로 주어진다. (1≤x_i, y_i≤1000, y_i와 y_i는 정수)

출력
제곱합오차가 최소인 a와 b를 한 줄에 공백으로 구분하여 출력하여라. 제곱합오차가 최소인 쌍 (a, b)가 여러 개인 경우는 입력으로 주어지지 않는다.


예제 입력
4
1 2
2 3
3 4
4 5

예제 출력
1 1
